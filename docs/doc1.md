## 프로젝트 개요 및 정의

| 항목 | 내용 |
| :--- | :--- |
| **프로젝트명** | AI Stock Genie (가칭: 지니의 투자 램프) |
| **목표** | 개인 투자자가 AI의 **명확한 추천 결론**을 통해 쉽고 캐주얼하게 투자 의사결정을 내릴 수 있도록 돕는 플랫폼 구축. |
| **톤 앤 매너** | **캐주얼, 친근함** 강조. 메인 액센트 컬러는 **밝은 주황색**. |

***

## 시장 분석 및 문제 해결 정의

### 💡 프로젝트 배경 및 필요성

개인 투자자는 정보 과부하와 편향된 시각에 직면해 있습니다. 전문성과 시간 부족으로 인해 **감정적이고 비합리적인 투자 결정**이 빈번하며, 기관 투자자와의 **정보 격차**를 해소할 친근하고 신뢰할 수 있는 도구가 필요합니다.

### ❓ 해결하고자 하는 핵심 문제와 해결 기능

| 핵심 문제 (The Pain Points) | 해결 기능 (Feature to Solve) |
| :--- | :--- |
| **1. "지금 사야 할까요?"에 대한 불확실성** | **AI 종합 추천 등급 제공** (UC-101) 및 **근거 요약 바로가기** |
| **2. 데이터의 난이도 및 지루함** | **재무/뉴스 감성 시각화** 및 **3가지 분석 탭** 정보 구조화 (UC-102, UC-104) |
| **3. 개인화된 분석 부재** | **조회 기록 기반 추천 팝업** 및 **전문가 추종 전략** (UC-106, UC-202) |
| **4. 종목 탐색의 어려움** | **관심순, 저평가순, 주의 종목 트렌드 제공** (UC-401, UC-402, UC-403) |

### ✅ AI 기반 핵심 차별화 전략 (How We Solve It)

| 구분 | 차별화 전략 | 해결 기능 목록 |
| :--- | :--- | :--- |
| **결론 제시** | **AI 종합 투자 등급:** 재무, 기술, 감성을 통합해 명확한 결론 제공. | **AI 종합 추천 등급** (주황색 강조), **근거 요약 및 바로가기** |
| **사용자 경험** | **캐주얼 & 직관성:** 데이터 시각화 및 정보 구조화로 피로도 감소. | **재무/뉴스 감성 시각화**, **3가지 상세 분석 탭** (재무/수급, AI 전략, 뉴스 감성) |
| **종목 발견** | **실시간 트렌드 제공:** 사용자의 시장 관심사를 빠르게 반영. | **관심순, 저평가순, 주의 종목 트렌드** |
| **개인화** | **행동 기반 추천:** 사용자 관심사를 반영한 맞춤형 정보 큐레이션. | **조회 기록 기반 추천 팝업**, **전문가 추종 전략** (유사도 점수) |
| **신뢰성** | **투명한 근거:** AI 판단의 배경과 기준을 세분화하여 공개. | **거래량 시그널 분석**, **전문가 전략 유사도** 표시 |

## 성공 측정 지표 (KPIs)

프로젝트의 성공을 측정할 비즈니스 관점의 핵심 지표입니다.

| 지표 (KPI) | 측정 목표 | 측정 방법 |
| :--- | :--- | :--- |
| **잔존율 (Retention)** | 월간 사용자 잔존율 30% 이상 유지. | 개인화된 추천 팝업 및 맞춤 전략 사용률. |
| **핵심 기능 사용률** | AI 종합 추천 등급 조회 수 90% 이상. | 종목 상세 페이지 진입 시 해당 API 호출 성공률. |
| **트렌드 사용률** | 메인 페이지 트렌드 캐러셀 클릭률 20% 이상. | UC-401/402/403 트렌드 리스트 클릭 수 모니터링. |
| **사용자 만족도** | 앱/웹 응답 시간 1초 미만. | 사용자 피드백 및 API Latency 모니터링. |
